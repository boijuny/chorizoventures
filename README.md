## Getting Started

### Prerequisites
- Node.js 18+
- Conda environment `guezvc`

### Development

1. **Activate environment:**
   ```bash
   conda activate guezvc
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open [http://localhost:3000](http://localhost:3000)**
